# SubCommands

WIP

Ideas:
- /sc muffle (on|off) (muffled chat, defaults to on)
- /sc blind (on|percentage|off) (black overlay with opacity, defaults to on)
- /sc mute (on|off) (disable chat input, defaults to on)
- /sc deaf (on|off) (hide chat messages, defaults to on)
- /sc drunk (on|off) (screen effect overlay, defaults to on)
- /sc bound (on|off) (disable move controls, defaults to on)
- /sc follow target (force follow, defaults to <t>)
- /sc confine (on|off) (can't leave room/house, defaults to on)
- /sc fpv (on|off) (force first person view, defaults to on)

Safe keybind by default on END
